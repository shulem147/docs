Voicemail

Plays voicemail instructions to leave a message: It's important to note that voicemails might end up in the spam folder. Voicemail to Email: Enter the email address in the destination field If you wish to send the emails to more than 1 email address at a time, simply separate each emails by a comma (email@gmail.com,email2@gmail.com) Skip Instructions: skipInstructions => true Copy Destination: When pressing 9 to copy the message while listening to a voicemail, the system will ask you for the IVR ID to copy to. When always copying from this voicemail mailbox to the same IVR destination, you could enhance the following: When pressing 9, the system will ask for the IVR ID, and the system will enter the IVR ID automatically, so you don't have to input it every time. copyDestination => IVR ID
