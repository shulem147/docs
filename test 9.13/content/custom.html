Custom

Set action for Customer sign up for Custom features under Snaptel or the following. To Add to a IVR File Recording: customAction => newRecord (add pin) Transfer to a Different Hotline in Our System: customAction => gotoAccount (put in IVR ID in the destination) Get Email Notification: noRecord => true (put in the email address in the destination) Custom Action - Bridge don't use anymore served as SIP and could use the same settings as transfer: customAction => bridge
